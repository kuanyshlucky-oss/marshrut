# Маршрут — подготовка к магистратуре

Учебный фронтенд-макет платформы подготовки к вступительным экзаменам в
магистратуру по группам образовательных программ (7M01–7M12).

## Возможности
- Каталог из 10 направлений с поиском по коду/названию
- Внутри направления — 4 экзаменационных предмета (2 общих + 2 профильных)
- По каждому предмету — список тем для подготовки
- Профильный тест по направлению (таймер, проходной балл 60%)
- Личный кабинет: личные данные, результаты тестов, избранные направления

## Технологии
Чистый HTML + CSS + JavaScript, без сборки. «Бэкенд» — имитация на
`localStorage` (данные хранятся в браузере посетителя, не на сервере).

## Запуск
Открыть `index.html` в браузере — сборка не требуется.
